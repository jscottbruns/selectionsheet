Add/Edit Lot

A lot is a location (or buildable lot) within a community. You can create and store an unlimited number of lots. Each lot created can be added to your running schedule. Lot information is stored and is always available for future reports.

Note: In order to create lots and running schedules...

...you must first create a community.

To add or edit a lot, do the following:

  1. Select My Info > My Active & Pending Lots. The My Active & Pending Lots screen displays.

  2. Click Add A New Lot or click Edit Lot for the lot you want to edit.

  3. Complete the fields as necessary.
    Note:     Fields with an *must be completed...

    ...in order to save the lot information.

  4. Click UPDATE or SUBMIT to save. The My Active & Pending Lots screen displays with the updated lot information.

Note: A lot remains in "pending" status until...

...you schedule the lot for construction.
