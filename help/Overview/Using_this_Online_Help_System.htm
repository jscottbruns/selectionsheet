Using this Online Help System

The SelectionSheet.Com Online Help system will guide you through the tasks necessary to schedule and manage your construction projects using all of the functions of SelectionSheet.Com.

Note: When you access this Online Help, if an information bar displays the following prompt: "To help protect your security, Internet Explorer has restricted this file from showing active content that could access your computer. Click here for options...", click the information bar, select Allow Blocked Content, and click Yes.

The following are some important features found in this Online Help:

  • Drop down text
    Throughout this Help system, drop down text is used to provide information on demand.
    When you see a note, simply click on the text and the note will expand. All notes should be read at least once.

    For example, click on the following text:
     Note:Click here to see this note...

    ...expand. Click the link again to collapse this expanded text.

All expanding text in a topic can opened or closed by clicking Show/Hide Drop Down Text on the Help System toolbar.

  • Links
    Links are included throughout the Help system. They provide a shortcut to related information or more task-specific instructions.

  • Toolbar
    This Help system includes the following buttons along the top of the Help screen:

    • Contents
      Displays the Contents pane on the left-hand side of the browser window. Books are represented by (or if opened). Topic links are represented by . Click a book to display its list of topics. Click a topic link to view it. .

    • Index
      Displays the Index pane on the left-hand side of the browser window. Type the first few letters of the word you are looking for or scroll through the list of index entries. Then, click the closest matching word to display the topic.

    • Search
      Displays the Search pane on the left-hand side of the browser window. Type the word you are searching for and click . A list of any topics containing the word displays. Click on a topic name to display the topic.

    • Glossary
      Displays the Glossary pane on the left-hand side of the browser window. Scroll through the list of glossary entries and select the word you want. The definition for the word displays below the list of glossary entries.

    • Show/Hide Drop Down Text
      Shows/hides all drop down text in the current topic. Use this button if you want to make sure you don't miss any important notes or information. Click to show all text. Click again to hide drop down text.
