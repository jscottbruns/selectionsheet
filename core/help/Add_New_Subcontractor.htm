Add/Edit Subcontractor

Use the My Subcontractor screen to add or edit subcontractors.

Note:You can also convert a contact to...

...a subcontractor by checking the Subcontractor checkbox on the Contact information screen and then specifying the trades associated with the contact.

To add or edit a subcontractor, do the following:

  1. Select My Info > My Subcontractors. The My Subcontractors screen displays.

  2. Do one of the following:

    • Click Add New Subcontractor. The Add A New Subcontractor screen displays.

    • Click Edit Sub for the Subcontractor you wish to edit. The Edit A Subcontractor screen displays.

  4. Complete the fields as necessary.
    Note:     Fields with an *must be completed...

    ...in order to save the subcontractor information.

  5. Click ADD or UPDATE to save. The My Subcontractors screen displays.
