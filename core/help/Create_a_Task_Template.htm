Create a New Task Template

For new task templates with a majority of new or different tasks, you can build a new template from scratch.

Note: When you create your new template from scratch...

...you will create each task through our template builder. If you can't finish all your tasks in one sitting, just save the template builder and you can pick up later. The template builder also saves your work as you go, so if you are interrupted you will never have to start over.

To create a new task template from scratch, do the following:

  1. Select My Info > My Task Templates. The My Task Templates screen displays.

  2. Click Create a New Task Template. The Create a New Task Template screen displays.

  3. Click Build your template from scratch.

  4. In the Template Name field, enter the name you want for this task template.

  5. In the Production Days field, enter the number of production days you want to include in this template.
    Note: Your construction time should...

    ...not include weekend or non work days. You can always adjust your construction time when you're inside the template builder. This is just an estimate.

  6. Click Load Template Builder. The Template Builder screen displays.

  7. You will now create the tasks to include in the template.
    Note:The Template Builder autosaves...

    ...depending on the setting in the AutoSave Every pull down list. Specify the autosave interval by selecting:

    • 5 Min (default)

    • 10 Min

    • 30 Min

    • 10 Seconds

    • Never

    When the specified amount of time elapses, the system saves any changes made and refreshes the screen.

  8. In the Insert New Task Box, in the Task Name field, enter a name for the task.

  9. In the Phase field, enter a number representing the production day on which you want this task to start.

  10. Use the Duration drop down list to specify the number of days this task should take.

  11. Use the Task Type drop down list to specify the type of task that will be performed.

    Below are the available options:

    • Labor (General Task)
      Use this option if you want to create a labor-related task. For example, an Installation task for installing Doors & Windows.

    • General Reminder
      Use this option if you want to create a reminder only task.

    • Delivery
      Use this option if the task involves delivery of materials. For example, the delivery Doors & Windows.

    • Delivery Reminder/Order Materials
      Use this option if you need to be reminded to schedule delivery or order materials.

    • Inspection
      Use this option if the task is for an inspection. For example, inspection of exterior structures.

    • Inspection Reminder
      Use this option if you need to be reminded that an inspection is needed or to plan for an inspection.

    • Appointment
      Use this option if you need to plan an appointment. For example, an appointment with the new home owner for a walk through.

    • Paperwork
      Use this option for paperwork tasks. For example, paperwork required for obtaining permits.

    • Other Task
      Use this option if the task does not fall into any of the above categories.

  12. Use the Parent Category drop down list to specify the category to which this task is most closely related.

  13. Use the Sub Tasks section to add sub tasks to this task. Select a sub task category, then enter a name, phase, and duration for the sub task. Sub tasks can be configured later.

  14. Click Insert Task to insert the task into the task template.

  15. (Optional) Repeat steps 7-14 as needed. When you are finished creating and moving your tasks within the template builder, you will create task relationships (critical paths).

  16. Click Create My Task Relationships. A prompt displays.

  17. If you are sure you want to proceed, click OK. You can not go back once you click OK. The Define Your Task Relationships screen displays.

    On this screen, you will define task relationships (critical paths). Your primary tasks created with the Template Builder are shown in the left column, with their production day in ().

  18. Starting with the first task, check the boxes that are prerequisites to the task. These are tasks that must be completed before the task in bold can occur. Then, click NEXT and repeat this step for each task.
     Note:Once you assign the task in bold its relationships...

    ...it will be grayed out. You can always edit and change your tasks relationships later.
