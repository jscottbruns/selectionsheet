Add/Edit a Community

A community is a location designated by name, city, county, state and zip code, and consists of one or more buildable lots. You can create and store an unlimited number of communities.

Note: In order to create lots and running schedules...

...you must first create a community.

To add or edit a community, do the following:

  1. Select My Info > My Communities. The My Communities screen displays.

  2. Do one of the following:

    • If no communities exist, proceed to Step 3.

    • Click Add A New Community or click the name of the community you wish to edit.

  4. Complete the fields as necessary.
    Note:     Fields with an *must be completed...

    ...in order to save the community information.

  5. Click UPDATE or SUBMIT to save. The My Communities screen displays with the updated community information.
