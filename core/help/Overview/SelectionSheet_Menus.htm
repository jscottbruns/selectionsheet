SelectionSheet.Com Menus

SelectionSheet.Com has a web-based graphical interface. All functions are accessible via pull down menus.

Menu bar

The menu (or tab) bar contains the following items:

  • HOME
    Use this menu to return to the SelectionSheet.Com main screen.

  • MY INFO
    Use this menu to add, configure, and view communities, active/pending lots, completed lots, subcontractors, and task templates.

  • SCHEDULES
    Use this menu to configure and view running schedules and appointments.

  • REPORTS
    Use this menu to configure and run reports.

  • CORRESPOND
    Use this menu to access the SelectionSheet.Com user forum, check and send email, maintain contacts, and contact SelectionSheet.Com, Inc.
