Schedule Construction

After you create a lot, it remains in "pending" status until you schedule the lot for construction.

To schedule a lot for construction, do the following:

  1. Select My Info > My Active & Pending Lots. The My Active & Pending Lots screen displays.

  2. Click Schedule Construction under the appropriate lot. The Schedule A Lot For Production screen displays.

  3. Use the Select Your Building Template drop down list to select the task template you want to use for this lot.

  4. Do one of the following:

    • Schedule your lot from start
      If you construction has not yet begun and you want to schedule this lot from the start, select a start date from the calender.

    • Schedule your lot midway through production
      If this lot is already in progress, choose a task that is occurring today. The associated running schedule will reflect this task and assume tasks in the template that should have occurred prior to this task have been completed.

  6. Click Schedule to confirm. The My Active & Pending Lots screen displays with the updated information.
