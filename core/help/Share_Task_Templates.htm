Share a Task Template

To share a task template with another SelectionSheet.Com user, do the following:

  1. Select My Info > My Task Templates. The My Task Templates screen displays.

  2. Select Share My Task Templates. The Share My Task Templates screen displays.

  3. Use the drop down list to select the task template you want to share.

  4. In the SelectSheet Username field, enter the user name or email address of the SelectionSheet.Com user you want to share this task template with.

  5. Click SUBMIT.
    Note:An email will be sent to...

    ...the specified user with a link to instructions on importing your task template into their account.
