Edit a Task Template

To edit a task template, do the following:

  1. Select My Info > My Task Templates. The My Task Templates screen displays.

  2. Select the task template you want to edit. The My Tasks screen displays.

  3. To change the template name, enter the new name in the Rename Task Template field and click Go.

  4. To add a task to this template, click Add A New Task and follow the prompts.

  5. To edit task detail, select a task from the list on the left-hand side of the screen. The Edit / Archive My Task summary screen displays.
    You can edit the task detail by doing the following:

    • To change the task name, click the Task Name tab, enter a new name in the Task Name field, and click UPDATE.

    • To change the task duration, click the Duration tab, enter a new duration, and click UPDATE.

    • To change the task phase (the day on which you want this task to start) click the Phase tab, find and select the day you want and click PREVIEW. Then, click DONE if you want to accept the changes.
      Note: If this task has post task relationships...

      ...those tasks defined as Post Task Relationships (Critical Path) will move according to your task. To move your task more freely, you can click Release Post Task Relationships.

    • To add sub tasks, click the Sub Tasks tab, enter the sub task information, and click UPDATE.
      Note: On the Sub Task screen you can...

      ...specify the name of the sub task and when the sub task should occur in relation to the main task (number of days before, number days after, or on same day as).

    • To add reminders to this task, click the Reminder tab and select the tasks you would like to link to this task. Then, click UPDATE.
      Note: Reminders allow you to...

      ...be notified when related tasks are completed. Multiple reminders can be tagged to one task. This allows you to confirm a group of tasks.

    • To add or change pre task relationships, click the Pre Task Relationships tab and click EDIT. Check the boxes that are prerequisites to the task and click UPDATE.
       Note:These are tasks that...

      ...be completed before the task in bold can occur.

    • To add or change post task relationships, click the Post Task Relationships tab and click EDIT. Check the boxes that must be completed after this task and click UPDATE.
