Access Discussion Forum

The SelectionSheet.Com discussion boards are included with your account access for you to communicate with other SelectionSheet.Com users. Each and every registered user has access to the same discussion boards.

To use the Discussion Forums, do the following:

  1. To access the discussion boards, select Correspond > Discussion Forum. The Discussion Forum screen displays.

  2. Select a category to view discussions.

  3. To start a new message topic (or thread) within a main topic, click ,enter your message and click Submit New Thread.

  4. To respond or to add you opinion to a thread while viewing a message, click ,enter your message and click Submit Reply.
